Обязать водителей общественного транспорта включать аварийную световую сигнализацию при посадке и высадке пассажиров

Часто при остановке общественного транспорта создаются трудности для водителей других автомобилей. Предлагаю для предупреждения участников дорожного движения обязать водителей общественного транспорта включать аварийную световую сигнализацию при посадке и высадке пассажиров.

Практический результат

Повышение безопасности участников дорожного движения.

Решение

Обязать водителей общественного транспорта включать аварийную световую сигнализацию при посадке и высадке пассажиров.
Для голосования вы должны быть авторизованы через ЕСИА
